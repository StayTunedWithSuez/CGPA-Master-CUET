## Download
Download the extension from the GitHub Release:  
[CGPA Master - CUET v1.0.0](https://github.com/StayTunedWithSuez/CGPA-Master-CUET/releases/download/v1.0.0/CGPA-Master-CUET.v1.0.0.zip)

## Installation Guide

1. Download the extension file from the download option above.  
2. Open your Chrome, Brave, Edge, or any Chromium-based browser.  
3. Click the three-dot menu at the top-right corner.  
4. Select **Extensions**, then click **Manage extensions**.  
5. Enable **Developer mode** at the top-right if it is turned off.  
6. Open your file manager, locate the downloaded file, and drag & drop it into the Extensions page.  
7. The extension will appear in the list—turn it on if it is off.  
8. Visit the CUET result publishing site, and you will see a new button named **"Analyze Your Result"**. Click it and enjoy!

## About this Extension

**CGPA Master - CUET** is a Chrome extension for students of **Chittagong University of Engineering and Technology (CUET)**. It allows complete result analysis directly on the CUET main result website.

### Features
- Calculate **overall CGPA**
- Show **course summary**
- Display **last term results**
- View **term-wise CGPA**
- Show **grades per subject**
- Generate **CGPA graphs**
- Calculate **target CGPA**
- Perform **“what-if” CGPA calculations** for improvement exams
